## About

Expense tracker web app created using cra, react-hooks, react-context-API, react-reducer-provider


## Demo 
[Your-Munim](https://kind-lumiere-fd918c.netlify.app/)


## Installation
* `git clone https://github.com/srpsatish/your-munim.git`
* `cd your-munim`
* `yarn`
* `yarn start`
